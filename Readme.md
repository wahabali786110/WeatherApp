# 🌤️ Simple Weather Tracker

Hi there. Welcome to my weather app. 

I built this project to practice my Python backend skills and learn how to pull live data from the internet. It is a simple, fast web application that lets you type in any city name and instantly get the current weather, temperature, and wind speed.

To keep the application fast and secure, I decided not to use heavy frontend JavaScript. Instead, the backend (built with **FastAPI**) does all the heavy lifting. It talks to the external weather service securely, and then uses **Jinja2** to build the HTML page directly on the server before sending it to your screen. 

For the design, I used **Tailwind CSS** to give it a clean, modern "frosted glass" look that works well on both phones and computers.

### 🛠️ Built With:
* **Python & FastAPI** (For the backend server)
* **Jinja2** (To render the HTML pages)
* **Tailwind CSS** (For the design and styling)
* **WeatherAPI** (For the live weather data)
